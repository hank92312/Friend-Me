## 專案目前進度 (Current Progress)
- **聯網架構已確立**：採用 **FastAPI (Python) + WebSockets** 實現即時同步。
- **房間系統完成**：實作了「創立房間碼 (6位數)」、「加入房間」、「複製代碼」以及「待機大廳」功能。
- **大廳同步機制**：自動產生隨機玩家名稱，透過伺服器推播 `player_list_updated` 實現所有玩家名單即時同步。
- **隊長輪替邏輯**：實現了自動輪流擔任隊長的邏輯。非隊長玩家會進入「等待隊長選題」畫面。
- **同步遊戲階段**：隊長選題與房主開始遊戲，都會透過伺服器廣播帶動全場同步換關。
- **Phase 3/4 優化**：支援反悔、多彩配對、答案數量過多時的自適應捲軸與字體縮放、以及跨輪次累計統計。

## 測試指南 (How to Test)
1. **啟動後端**：`cd backend` -> `.\venv\Scripts\activate` -> `uvicorn main:app --reload`
2. **啟動 Godot**：同時開啟兩個 Godot 視窗 (F5) 進行「雙開測試」。
3. **驗證流程**：
   - 視窗 A (房主)：創立房間 -> 複製代碼。
   - 視窗 B (隊員)：輸入代碼加入 -> 兩邊名單應同步。
   - 視窗 A：點擊「開始遊戲」 -> 兩邊同步進入選題。
   - 視窗 A：選題 -> 兩邊同步進入回答。

**接手建議**：先進行雙開連線測試，確認大廳與選題同步無誤後，開始實作「答案內容的即時同步」。
